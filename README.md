## How to Use

1. **Add your profile image**: Replace `profile-placeholder.jpg` with your actual photo
2. **Add your CV**: Add a `cv.pdf` file for the download link
3. **Customize content**: Edit the HTML file to add your information
4. **Update GitHub links**: Add links to your project repositories
5. **Deploy**: Upload all files to your web hosting or GitHub Pages

## Customization Tips

### Adding More Projects


### Updating Skills
Edit the skill tags in the Technical Skills section:
```html
<span class="skill-tag">Your Skill</span>
```

### Changing Colors
Edit the CSS variables in `styles.css` to match your preferred color scheme.
